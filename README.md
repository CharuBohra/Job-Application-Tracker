
Job Application Tracker is a modern dashboard-style web application that helps users manage and track their job applications efficiently. It allows adding, editing, deleting, and filtering job entries while providing a clear overview of application statuses through summary cards and a searchable table. The project is built using React with Tailwind CSS for a clean and responsive UI, and uses localStorage for data persistence.

## ✅ Key Features
- Dashboard summary cards for application statistics
- Add, edit, and delete job applications
- Status-based filtering and company search
- Colored status badges for quick visual tracking
- Persistent data using browser localStorage
- Responsive and modern UI with Tailwind CSS
- Clean component-based architecture

## ✅ Tech Stack
- React.js
- JavaScript (ES6+)
- Tailwind CSS
- React Icons
- LocalStorage

This project was built to practice real-world frontend development patterns including component structuring, state management, and UI design.
